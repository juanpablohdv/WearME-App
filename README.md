# WearME APP

WearME is a mobile application designed to interface with a physical wearable prototype of the same name. Together, they comprise the project "Design and Development of a Wearable Device for the Acquisition and Visualization of Physiological Variables Related to Respiratory Diseases." I contributed to the development of this project, which was presented in May 2026 as the final capstone project for the Bachelor's Degree in Electronic Engineering.

The file WearMEAPP.zip contains the packaged application, ready to be extracted and installed. The WearMEPROJECT folder contains the complete source code of the application
